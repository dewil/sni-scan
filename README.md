# SNI Scan (Local Subnet)

Небольшой Python-скрипт для сканирования локальной подсети, поиска открытого `443/tcp` и извлечения доменных имен из TLS-сертификатов (CN/SAN) с проверкой DNS -> IP.

## Что делает

- определяет текущий локальный IPv4 адрес;
- сканирует сеть с маской (по умолчанию `/24`);
- проверяет доступность `443/tcp` на хостах;
- извлекает `CN` и `SAN` из сертификата;
- проверяет, совпадают ли DNS A-записи доменов с IP хоста;
- сохраняет отчет в `.md`.

## Требования

- Python `3.9+`
- Linux/macOS/Windows (без внешних зависимостей)

## Быстрый старт

```bash
python3 sni-scan.py -o report.md
```

## Примеры

```bash
# Скан по умолчанию (/24)
python3 sni-scan.py -o report.md

# Скан другой маски (/26)
python3 sni-scan.py -m 26 -o report_26.md
```

## Параметры

- `-o, --output` — путь к markdown-отчету
- `-t, --timeout` — таймаут на хост (сек)
- `-w, --workers` — число параллельных потоков
- `-m, --mask` — маска подсети (`0..32`, по умолчанию `24`)

## Выходной отчет

В отчете есть:

- IP хоста;
- статус TLS;
- `CN` и `SAN` (кандидаты SNI);
- колонка `DNS -> IP match` (`yes/no/partial/-`);
- служебные заметки по каждому хосту.

## Важно

Используйте только в своей сети или при наличии разрешения на сканирование.
